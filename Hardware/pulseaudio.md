# Pulseaudio

Install pulseaudio and pulseaudio-alsa
```
sudo pacman -S pulseaudio
sudo pacman -S pulseaudio-alsa
sudo pacman -S pulsemixer

pulseaudio --check
pulseaudio --start

pulsemixer
```

Check alsamixer after you install them
as outputs might get muted, unmute them

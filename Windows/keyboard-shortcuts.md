# Keyboard shortcuts

Ctrl + Shift + Escape  => Task Manager
Ctrl + R => Run Window

### Virtual Desktops

Super + Tab => Check VDs
Ctrl + Super + LeftArrow => Switch to VD on Left
Ctrl + Super + RightArrow => Switch to VD on Right
Ctrl + Super + D => Create New VD
Ctrl + Super + F4 => Kill Current VD
